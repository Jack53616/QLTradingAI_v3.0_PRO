# QL Trading AI - Frontend Requirements

## Design
- Professional Binance-style dark theme
- Glassmorphism effects
- Smooth animations
- Arabic RTL support
- Responsive design

## Features

### 1. Activation Gate
- Video background (bg.mp4)
- Name input field
- Email input field
- Key input field with smart extraction
- Confirm button (MUST work!)
- Buy key button (WhatsApp link)
- Toast notifications

### 2. Main App
- Balance card with VIP badge
- Real-time crypto prices from CoinGecko
- Activity feed on home page
- Quick action buttons
- Stats cards (Day, Month, Subscription)

### 3. Tabs
- Home/Wallet
- Markets (Gold, Silver, BTC, ETH)
- Trades (with SL/TP)
- Withdraw (crypto methods)
- Requests
- Support

### 4. APIs
- /api/auth - Authentication
- /api/activate - Activation with key
- /api/wallet - Wallet data
- /api/activity - Activity feed
- /api/markets - Market prices
- /api/withdraw - Withdrawal requests

## Technical Requirements
- Clean, organized code
- Proper event listeners
- Error handling
- Loading states
- Console logging for debugging
- No syntax errors
- All features must work 100%
